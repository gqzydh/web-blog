前端常用插件：
CSSLint：可以高亮 CSS 错误及代码质量问题

Beautify：让HTML、CSS、JavaScript 等代码格式化一致，从而提升可读性
eqFTP： WEB 主机传输和同步文件， 同 FTP 或者 SFTP一样 
QuickDocsJS： JS 帮助文档 
选中位置，使用快捷键 command + K（mac），ctrl + K （windows），可以快速查看 JavaScript 的帮助文档。
